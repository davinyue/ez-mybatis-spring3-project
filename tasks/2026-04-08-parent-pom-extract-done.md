# 提取父POM公共配置

## 修改背景
- 多个子模块 `pom.xml` 中存在重复的项目元数据、Java 编译属性和发布插件配置。
- 需要将可复用配置提取到父 `pom`，降低维护成本，同时不能影响发布到 Maven Central。

## 需求
- 分析并提取各子 `pom` 可上提到父 `pom` 的公共配置。
- 不能影响 Maven Central 发布相关配置与行为。
- 涉及 `mybatis-plus` 的依赖与版本配置不提取。
